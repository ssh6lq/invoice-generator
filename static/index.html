영수증 이미지로
한 번에 비용 청구서를 완성하세요

처음이신가요? 사용 방법 보기
1.
기초정보 입력 — 청구항목·소속부서·성명을 입력해요.
2.
영수증 사진 업로드 — 청구할 영수증을 한 번에 첨부할 수 있어요.
3.
이미지 분석 — 날짜·사용처·금액을 자동으로 추출해요.
4.
검토 후 생성 — 추출된 결제 내역으로 양식을 자동으로 채워 청구서를 생성해요.
1기초정보 입력

청구항목·소속부서·성명을 입력하세요. 입력된 정보는 청구서에 반영됩니다.

2영수증 이미지 업로드

청구할 영수증 사진을 업로드합니다. (여러 파일 첨부 가능)

영수증 사진을 끌어다 놓거나 선택하세요
JPG · PNG · WEBP · BMP
파일 선택
3영수증 분석

영수증의 날짜·사용처·금액을 자동으로 추출합니다.

4검토 / 수정

분석으로 추출된 내역으로 양식을 작성합니다. '+ 행 추가'로 직접 입력할 수도 있습니다.

⚠️ AI가 자동으로 추출한 정보로 오인식 될 수 있습니다. 실제 영수증과 대조하여 최종 비용청구서를 생성해 주세요.
일괄 채우기
5비용청구서 생성

매크로 검토로 필수항목·형식·당해연도·청구금액·식대 한도를 점검한 뒤, 통과하면 비용청구서 시트를 완성해 다운로드합니다.

다운로드한 청구서에서 매크로가 차단되나요? — 최초 1회 설정

이 서버에서 받은 비용청구서(.xlsm)는 Windows가 외부 파일로 보고 매크로를 막을 수 있어요. 아래를 PC마다 한 번만 설정하면 이후엔 매크로가 바로 동작합니다.

1.
윈도우 검색 → 인터넷 옵션 실행
2.
[보안] 탭 → [로컬 인트라넷] 선택 → [사이트] → [고급]
3.
"영역에 웹 사이트 추가"에 이 주소를 넣고 추가:
…
4.
닫기 → 확인. 이후 받은 파일을 열 때 [콘텐츠 사용]만 누르면 됩니다.

※ '신뢰할 수 있는 사이트'로 등록할 땐 'https 필요' 체크를 해제하세요. 서버 IP가 고정일 때 권장합니다.

근태현황 파일로
한 번에 초과근무신청서를 완성하세요

처음이신가요? 사용 방법 보기

그룹웨어의 월간 근태현황 파일을 첨부하면, 승인된 초과근무 날을 자동으로 찾아 드려요.

1.
월간 근태 현황 파일을 업로드 — 승인받은 초과근무 날을 탐지해요.
2.
대체휴무 반영 — 대체휴무로 쓴 시간은 신청 대상에서 제외해요.
3.
초과근무 신청서 생성 — 확정된 시간으로 신청서를 생성해요.
1근태현황 업로드

월간 근태현황(.xlsx)을 올리면 초과근무 대상일을 자동으로 탐지합니다.

월간 근태현황 파일을 끌어다 놓거나 선택하세요
XLSX
파일 선택
2초과근무 내역 검토

근태현황 파일에서 자동으로 추출한 초과근무 대상일이에요.
대체휴무를 사용한 날은 대체휴무를 O로 변경 후, 시간(예: 1.5)을 입력하세요. 입력한 만큼 신청 시간에서 자동 차감됩니다.

3초과근무신청서 생성

승인된 초과 근로 시간으로 비용 청구서를 생성합니다.
