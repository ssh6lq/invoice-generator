비용청구서를 완성하세요

1기초정보 입력

청구항목·소속부서·성명을 입력해 비용 청구서를 초기화합니다. 입력한 정보는 청구서 양식 상단에 그대로 반영됩니다.

2영수증 이미지 업로드

청구할 영수증 사진을 업로드합니다. (여러 장 동시 첨부 가능, 추가 첨부 가능)

☁︎
영수증 사진을 끌어다 놓거나 선택하세요
JPG · PNG · WEBP · BMP
파일 선택
3영수증 분석

영수증의 날짜·사용처·금액을 자동으로 추출합니다. (이미 분석한 사진은 건너뜀)

4검토 / 수정

분석으로 추출된 내역으로 양식을 작성합니다. '+ 행 추가'로 직접 입력할 수도 있습니다.

일괄 채우기
5비용청구서 생성

작성한 내용으로 비용 청구서를 생성합니다.

근태현황 파일로
연장근무신청서를 자동 작성하세요

1근태현황 업로드

월간 근태현황(.xlsx)을 올리면 연장근무 대상일을 자동으로 찾아 보여줍니다.

☁︎
근태현황 파일을 끌어다 놓거나 선택하세요
XLSX
파일 선택
2대체휴무 입력

기본은 X(대체휴무 미지급) → 근무시간 전체가 신청됩니다. 받은 날만 O로 바꾸고 대체휴무시간(예: 1:30 또는 1.5)을 넣으면 그만큼 차감됩니다.

대체휴무지급 일괄
