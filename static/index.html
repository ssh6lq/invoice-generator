비용청구서를 완성하세요

1기초정보 입력

청구서에 들어갈 기본 정보예요. 성명을 넣으면 본인확인이 '서명완료'로 처리됩니다.

2영수증 이미지 업로드

여러 장을 한 번에 올릴 수 있어요. 분석한 뒤 사진을 더 추가해도 됩니다.

☁︎
영수증 사진을 끌어다 놓거나 선택하세요
JPG · PNG · WEBP · BMP
파일 선택
3영수증 분석

새로 추가한 영수증만 분석해 기존 목록에 덧붙입니다. (이미 분석한 사진은 건너뜀)

4검토 / 수정

인식 결과를 확인하고 필요하면 고치세요. 목적·결제방식을 선택하면 한도가 적용됩니다.

일괄 채우기
5비용청구서 생성

생성하면 지금 접속한 기기(내 PC)로 다운로드돼요. 브라우저가 지원하면 저장 위치를 직접 고를 수 있습니다.

이 서버 PC 폴더에 직접 저장 (관리자용 · 매크로 경고 없음)
서버(이 PC)에서 작업할 때만 쓰세요. 원격 접속자는 위 '다운로드'를 사용하세요.

근태현황 파일로
연장근무신청서를 자동 작성하세요

1근태현황 업로드

월간 근태현황(.xlsx)을 올리면 연장근무 대상일을 자동으로 찾아 보여줍니다.

☁︎
근태현황 파일을 끌어다 놓거나 선택하세요
XLSX
파일 선택
2대체휴무 입력

기본은 X(대체휴무 미지급) → 근무시간 전체가 신청됩니다. 받은 날만 O로 바꾸고 대체휴무시간(예: 1:30 또는 1.5)을 넣으면 그만큼 차감됩니다.

대체휴무지급 일괄
